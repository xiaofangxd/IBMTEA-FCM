# IBMTEA-FCM
We provided matlab implementations for Learning Large-scale Fuzzy Cognitive Maps Using an Evolutionary Many-task Algorithm.

## Getting started

- Clone this repo:

```
git clone https://github.com/xiaofangxd/IBMTEA-FCM.git

```

- Install MATLAB.

### Running

- Running main.m. 

## Citation
